# An Almost Complete Guide to Far Cry 2 Modding

:::info[Reproduced with attribution]
This guide is reproduced **verbatim** from ["An Almost Complete Guide to Far Cry 2
Modding"](https://www.nexusmods.com/farcry2/mods/299) ([ModDB
mirror](https://www.moddb.com/mods/an-almost-complete-guide-to-far-cry-2-modding)), created and
maintained by **Boggalog**, originally published as a public Google Doc. It's the community's
closest thing to an SDK/reference manual for Far Cry 2 modding. Split here into one page per section
for navigability — the wording is unchanged from the source.
:::
